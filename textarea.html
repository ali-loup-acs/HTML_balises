<textarea>

Définition :

La balise <textarea> permet de définir une zone de saisie de texte multi-lignes. L’utilisateur/internaute peut saisir des données dans ce champ.

Exemple :

<textarea>
Texte affiché dans la balise textarea,
Texte affiché dans la balise textarea,
Texte affiché dans la balise textarea.
</textarea>
