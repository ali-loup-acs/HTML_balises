<img>

Définition :

La balise <img> permet d’insérer une image dans la page. La balise <img> est une balise simple, elle n’a pas de balise fermante. Plusieurs types de format peuvent être utilisés, à savoir JPEG GIF, y compris les GIFs animés PNG, APNG, SVG, BMP, BMP ICO, PNG ICO.

Exemple :

<img src="mon-image.jpg"/>
